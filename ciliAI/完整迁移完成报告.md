## 📚 API 文档

### 邀请码 API
```javascript
import { inviteCodeAPI } from '@/routes/api/ruoyi-api'

// 验证邀请码
const result = await inviteCodeAPI.validate(code, userId)

// 生成邀请码
const codes = await inviteCodeAPI.generate({
  count: 10,
  expiresAt: '2027-04-16',
  maxUses: 1
}, token)
```

### 算力 API
```javascript
import { powerAPI } from '@/routes/api/ruoyi-api'

// 获取算力
const power = await powerAPI.getByUserId(userId, token)

// 充值
await powerAPI.recharge(userId, amount, token)

// 消费
await powerAPI.consume(userId, amount, token)
```

### AI 生成 API
```javascript
import { aiAPI } from '@/routes/api/ruoyi-api'

// 文生图
const task = await aiAPI.text2img({
  prompt: '一只可爱的猫咪',
  style: 'anime'
}, token)

// 查询状态
const status = await aiAPI.getTaskStatus(task.taskId, token)
```
