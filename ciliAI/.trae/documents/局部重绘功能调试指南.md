## 常见问题排查

### 问题1：后端服务没有重启
**症状**：修改代码后仍然出现相同的错误

**解决方案**：
1. 停止后端服务（Ctrl+C）
2. 确认代码已保存
3. 重新启动后端服务

### 问题2：Pillow库未安装
**症状**：后端日志显示 "ModuleNotFoundError: No module named 'PIL'"

**解决方案**：
```bash
pip install Pillow
```

### 问题3：Mask转换失败
**症状**：后端日志显示 "Mask转换失败"

**排查步骤**：
1. 检查日志中的具体错误信息
2. 检查mask base64数据是否完整
3. 确认mask图像格式是否正确

### 问题4：火山引擎API调用失败
**症状**：后端日志显示 "API提交响应: code!=10000"

**排查步骤**：
1. 查看API返回的完整响应
2. 检查错误信息
3. 确认API参数是否正确
4. 检查网络连接

### 问题5：前端发送的数据格式错误
**症状**：后端日志显示数据长度异常

**排查步骤**：
1. 检查前端日志中的base64长度
2. 确认图片是否正确加载
3. 检查mask是否正确生成

## 技术细节

### Mask转换算法
1. **输入**：RGBA或RGB格式的PNG图像（带透明通道）
2. **处理**：
   - 移除base64前缀
   - 解码base64为二进制数据
   - 用PIL打开图像
   - 转换为RGB（去除透明通道）
   - 转换为灰度图（L模式）
3. **输出**：单通道灰度PNG图像

### API调用参数
```json
{
  "req_key": "jimeng_image2image_dream_inpaint",
  "binary_data_base64": [
    "原图base64",
    "mask灰度图base64"
  ],
  "prompt": "编辑提示词"
}
```
