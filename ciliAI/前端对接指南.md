### 步骤2：修改登录页面
在登录页面中添加邀请码验证：
```javascript
import { authAPI, inviteCodeAPI } from './routes/api/ruoyi-api'

// 登录时验证邀请码
async function handleLogin() {
  // 1. 验证邀请码
  const inviteResult = await inviteCodeAPI.validate(inviteCode, userId)
  if (!inviteResult.success) {
    alert('邀请码无效')
    return
  }

  // 2. 登录
  const loginResult = await authAPI.login(username, password)
  if (loginResult.code === 200) {
    // 保存 token
    localStorage.setItem('token', loginResult.token)
    // 跳转首页
    router.push('/')
  }
}
```

## ⚠️ 注意事项

1. **Token 传递**：所有需要认证的请求都需要在 Header 中传递 Token
   ```javascript
   headers: {
     'Authorization': `Bearer ${token}`
   }
   ```

2. **错误处理**：API 返回的是 RuoYi 标准格式
   ```javascript
   // 成功：{ code: 200, msg: '操作成功', data: {...} }
   // 失败：{ code: 500, msg: '错误信息' }
   ```

3. **邀请码验证**：在用户注册/登录前先验证邀请码

4. **算力检查**：AI 生成前检查用户算力是否足够
