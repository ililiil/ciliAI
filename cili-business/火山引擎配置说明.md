# 火山引擎 API 配置说明

## 📋 配置文件位置

在 `application.yml` 中添加以下配置：

```yaml
# 火山引擎配置
volcano:
  access-key: your_access_key_here
  secret-key: your_secret_key_here
  endpoint: open.volcengineapi.com
  region: cn-north-1
```

---

## 🔑 获取 API 密钥

1. 访问火山引擎控制台：https://console.volcengine.com/
2. 注册/登录账号
3. 创建 Access Key：
   - 进入「密钥管理」页面
   - 创建 Access Key（AK）和 Secret Key（SK）
4. 复制密钥到配置文件中

---

## 📡 支持的 API 接口

### 1. 文生图（text2img）
```java
volcanoEngineService.textToImage(
    prompt,              // 正向提示词
    negativePrompt,     // 负向提示词（可选）
    params              // 其他参数
);
```

**参数示例**：
```json
{
  "prompt": "a beautiful sunset over the ocean",
  "negative_prompt": "blurry, low quality",
  "style": "anime",
  "size": "1024*1024",
  "num": 1
}
```

### 2. 图生图（img2img）
```java
volcanoEngineService.imageToImage(
    imageUrl,           // 参考图片URL
    prompt,             // 正向提示词
    params              // 其他参数
);
```

### 3. 视频生成（video）
```java
volcanoEngineService.generateVideo(
    imageUrl,           // 图片URL
    params              // 其他参数
);
```

**参数示例**：
```json
{
  "duration": 5,
  "resolution": "720p"
}
```

### 4. 智能超清（enhance）
```java
volcanoEngineService.enhanceImage(imageUrl);
```

### 5. 局部重绘（inpaint）
```java
volcanoEngineService.inpaint(
    imageUrl,           // 图片URL
    maskUrl,            // 蒙版URL
    prompt              // 提示词
);
```

### 6. 动作模仿（motion_clone）
```java
volcanoEngineService.motionClone(
    imageUrl,           // 人物图片URL
    videoUrl            // 动作视频URL
);
```

### 7. 数字人（digital_human）
```java
volcanoEngineService.digitalHuman(
    imageUrl,           // 数字人图片URL
    script              // 台词脚本
);
```

## ⚠️ 注意事项

1. **密钥安全**：不要将密钥提交到版本控制系统
2. **错误处理**：所有 API 调用都包含异常处理
3. **异步任务**：大多数任务都是异步的，需要轮询查询状态
4. **频率限制**：注意 API 调用频率限制
5. **成本控制**：设置算力余额提醒，避免超额使用

---

## 🛡️ 安全建议

### 1. 环境变量
```bash
export VOLCANO_ACCESS_KEY=your_access_key
export VOLCANO_SECRET_KEY=your_secret_key
```

### 2. 配置中心
使用配置中心（如 Nacos、Apollo）管理密钥

### 3. 密钥轮换
定期更换 Access Key 和 Secret Key
