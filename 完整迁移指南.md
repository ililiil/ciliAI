### 方法二：直接进入第五阶段

#### 1. 对接用户端前端

修改 `ciliAI/ciliAI/src/utils/api.js`：

```javascript
const BASE_URL = 'http://localhost:8080'

export const api = {
  // 认证
  login: (data) => axios.post(`${BASE_URL}/auth/login`, data),
  register: (data) => axios.post(`${BASE_URL}/cili/auth/register`, data),
  
  // 邀请码
  validateInviteCode: (code) => axios.post(`${BASE_URL}/system/invitecode/validate`, { code }),
  
  // 算力
  getPower: (userId) => axios.get(`${BASE_URL}/cili/power/${userId}`),
  
  // AI 生成
  text2img: (data) => axios.post(`${BASE_URL}/cili/ai/text2img`, data),
  img2img: (data) => axios.post(`${BASE_URL}/cili/ai/img2img`, data),
  video: (data) => axios.post(`${BASE_URL}/cili/ai/video`, data),
}
```

#### 2. 配置前端代理

修改 `vite.config.js`：

```javascript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true,
    }
  }
}
```

## 🆘 遇到问题？

### 常见问题

#### 1. 后端启动失败
```bash
# 检查端口占用
netstat -ano | Select-String ":8080"

# 检查 MySQL
docker ps | Select-String mysql

# 检查 Redis
docker ps | Select-String redis
```

#### 2. 前端无法访问
```bash
# 检查前端进程
netstat -ano | Select-String ":80"

# 重启前端
cd d:/fangtang/RuoYi-Vue-master/ruoyi-ui
npm run dev
```

#### 3. 数据库连接失败
```bash
# 检查 MySQL 容器
docker logs ciliai-mysql

# 重启 MySQL
docker restart ciliai-mysql
```
