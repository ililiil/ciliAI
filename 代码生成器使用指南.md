## 📝 生成代码步骤

### 第一步：配置数据库连接

1. 点击左侧菜单 **"代码生成"**
2. 点击 **"导入"** 按钮
3. 选择需要生成代码的业务表
4. 点击 **"下一步"**

### 第二步：配置生成信息

填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| **模板类型** | 单表/树表 | 单表 |
| **生成方式** | zip压缩包/自定义路径 | 自定义路径 |
| **业务名称** | 业务英文名 | invitecode |
| **模块名称** | 模块英文名 | system |
| **模块描述** | 中文描述 | 邀请码管理 |
| **作者** | 作者名称 | admin |
| **包名** | Java包路径 | com.ruoyi.system |
| **路由地址** | 前端路由 | /invitecode |
| **菜单路径** | 菜单路由 | system/invitecode |

### 第三步：生成代码

1. 点击 **"生成代码"** 按钮
2. 等待代码生成完成
3. 下载生成的代码压缩包

## 📂 生成的文件结构

生成的代码包含以下文件：

```
├─cili                   # 业务模块名
│  ├─controller
│  │  └─CiliXxxController.java    # 控制器
│  ├─domain
│  │  └─CiliXxx.java             # 实体类
│  ├─mapper
│  │  ├─CiliXxxMapper.java      # Mapper接口
│  │  └─CiliXxxMapper.xml        # Mapper XML
│  ├─service
│  │  ├─ICiliXxxService.java    # 服务接口
│  │  └─impl
│  │     └─CiliXxxServiceImpl.java  # 服务实现
│  └─ruoyi-ui
│     └─src
│        └─views
│           └─system
│              └─ciliXxx
│                 └─index.vue     # Vue页面
```

## 🔧 代码整合步骤

### 1. 复制 Java 文件

将生成的文件复制到对应目录：

```bash
# 复制到 ruoyi-system 模块
src/main/java/com/ruoyi/
├── system/
│   └── controller/
│       └── CiliXxxController.java
│   └── domain/
│       └── CiliXxx.java
│   └── mapper/
│       ├── CiliXxxMapper.java
│       └── CiliXxxMapper.xml
│   └── service/
│       ├── ICiliXxxService.java
│       └── impl/
│           └── CiliXxxServiceImpl.java
```

### 2. 复制 Vue 文件

```bash
# 复制到 ruoyi-ui
src/views/system/ciliXxx/index.vue
```


## ⚠️ 注意事项

### 1. 表名前缀
生成的代码会去掉 `cili_` 前缀，需要在代码中手动添加。

### 2. 字段类型
确保数据库字段类型与 Java 类型对应：
- `bigint` → `Long`
- `varchar` → `String`
- `decimal` → `BigDecimal`
- `datetime` → `Date`
- `tinyint` → `Integer`
- `text` → `String`

### 3. 业务逻辑
生成的代码只包含基础 CRUD，需要根据业务需求添加：
- 邀请码生成逻辑
- 算力计算逻辑
- 火山引擎 API 调用
- 权限验证
