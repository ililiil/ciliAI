## 🔧 如果启动失败

### 问题 1：端口被占用
```powershell
# 查找占用端口的进程
netstat -ano | Select-String ":8080"

# 结束进程
taskkill /PID <PID> /F
```

### 问题 2：数据库连接失败
```powershell
# 检查 MySQL 是否运行
docker ps | Select-String mysql

# 如果未运行，启动
docker start ciliai-mysql
```

### 问题 3：权限不足
```powershell
# 以管理员身份运行 PowerShell
# 右键 PowerShell -> 以管理员身份运行
# 然后执行启动命令
```
