# 快速入门

> 来源: https://www.volcengine.com/docs/85621/1995636

---

# 步骤一: 注册账号
如还未持有火山引擎账户，请点击【[立即注册](https://console.volcengine.com/auth/signup?redirectURI=//www.volcengine.com/)】，注册成为火山用户。

# #步骤二: 登录+实名认证

* 请点击【[立即登录](https://console.volcengine.com/auth/login?redirectURI=//www.volcengine.com/)】，登录火山引擎控制台

 
* 如未进行实名认证，点击控制台任意服务的任意button会弹出弹窗，请点击【前往认证】，进行实名认证后再开通服务

# 步骤三: 进入智能视觉控制台

## 入口一
即梦AI落地页：https://www.volcengine.com/product/jimeng，点击【立即开通】，进入智能视觉控制台

## 入口二
进入火山引擎控制台点击右上角控制台导航，在智能视觉分类下，点击【图像生成大模型】/【即梦AI】进入智能视觉控制台

## 入口三
直接从下方链接进入控制台
智能视觉控制台链接：https://console.volcengine.com/ai/overview

# 步骤四:开通能力
选择想要开通的服务，点击该服务对应操作栏【开通服务】，弹出服务开通弹窗

 

## 开通免费试用
选中【免费试用】选项，点击去开通，进入确认开通页面，点击【确认开通】

## 开通正式调用
选中【正式调用】选项，点击去开通，进入确认开通页面，点击【确认开通】

# 步骤五：获取AK/SK
在调用火山引擎智能视觉控制台的各个能力之前，确保您已生成访问密钥（AccessKey）。AccessKey包括AccessKeyID（AK）和AccessKeySecret（SK），其中AccessKeyID用于标识用户，AccessKeySecret是用来验证用户的密钥，请妥善保管。
**获取方式：** 点击右上角账号下拉列表，选择【API访问密钥】，点击【新建密钥】按钮，可获取AK/SK，可以此为凭证调用已经开通能力的API接口。

**建议您仔细了解使用主账号密钥的风险，您也可以选择忽视此提示，选择继续创建。** 

# 步骤六：启动开发
**接入文档入口示例一：** 点击该服务对应操作栏【接入文档】，看相关API接入文档进行服务调用

**接入文档入口示例二：** [官网文档中心](https://www.volcengine.com/docs)部分，选择【人工智能与算法】分类，选择对应分类的产品，可跳转至该API产品接口文档页面进行查看并调用
为协助您快速调用API接口，火山引擎为用户提供快速接入服务端SDK示例，该SDK封装了网关签名与API调用。目前，SDK支持Python、Golang、PHP等常用语言，您可以选择对应语言类目下的视觉技术能力SDK进行接入。可通过进入[AI中台公用文档-快速接入](https://www.volcengine.com/docs/6444/69729)查看。

# 查看已经开通的免费/正式服务
进入导航【概览】页面，点击服务管理，进入服务管理页面，可查看所有开通的服务

# 增购QPS/资源包

## 入口一
导航栏找到模型广场下的服务分类并点击服务对应分类，选择已经开通服务的操作栏，点击【计费购买】，购买资源包/QPS

## 入口二
进入【概览】页面，页面下方找到【已开通API列表】部分，选择想要购买的产品服务名，点击该服务名对应操作栏的【计费购买】

# 查看用量
导航栏找到【系统管理】，点击进入【用量统计】页面，可查看已开通能力API接口的调用量、QPS/并发、错误量、耗时

 

# 实例管理
导航栏找到系统管理，点击进入【实例管理】页面

## 手动续费
找到想要续费的实例，点击【续费】

 

## 开通自动续费
找到想要开通自动续费的实例，点击【开通自动续费】

 

## 关闭自动续费
找到想要关闭自动续费的实例，点击【关闭自动续费】

 

## **更多续费**
请点击【续费管理】，跳转续费管理界面
