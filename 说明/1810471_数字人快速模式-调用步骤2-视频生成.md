# 数字人快速模式-调用步骤2-视频生成

> 来源: https://www.volcengine.com/docs/85621/1810471

---

# 接口简介
根据用户上传的单张**图片**+**音频**，生成该图片对应的视频效果。OmniHuman数字人模型支持输入任意画幅包含人物的图片，结合音频，生成生动高质量的视频结果。人物的情绪、动态与音频具有强关联性。同时模型对动漫、卡通等形象等有一定的泛化性，可广泛应用于内容表达、唱歌和演绎等场景。
**强烈建议**：请先执行**调用步骤1：主体识别**，确保输入图中存在人、类人、拟人等主体。

## 提交任务

### **提交任务请求参数**

#### **Query参数**
:::tip 拼接到url后的参数，示例：[https://visual.volcengineapi.com?Action=CVSubmitTask&Version=2022-08-31](https://visual.volcengineapi.com?Action=CVSubmitTask&Version=2022-08-31)

:::
|参数 |类型 |**可选/必选** |说明 |
|---|---|---|---|
|Action |string |必选 |接口名，取值：**CVSubmitTask** |
|Version |string |必选 |版本号，取值：2022-08-31 |

#### **Header参数**
:::warning
本服务固定值：**Region为cn-north-1，Service为cv**
:::
主要用于鉴权，详见 [公共参数](https://www.volcengine.com/docs/6369/67268) - 签名参数 - 在Header中的场景部分

#### **Body参数**
:::warning
业务请求参数，放到request.body中，MIME-Type为**application/json**

:::
|参数 |类型 |**可选/必选** |说明 |
|---|---|---|---|
|req_key |string |必选 |服务标识|
| | | |取固定值: **jimeng_realman_avatar_picture_omni_v2** |
|image_url |string |必选 |图片URL（需公网可访问） |
|audio_url |string |必选 |音频URL链接（需公网可访问） |
