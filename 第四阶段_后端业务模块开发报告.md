## 🔧 代码集成步骤

### 1. 复制邀请码模块代码

将以下文件复制到 `ruoyi-system` 模块：

```bash
# src/main/java/com/ruoyi/system/
cp CiliInviteCode.java domain/
cp CiliInviteCodeMapper.java mapper/
cp ICiliInviteCodeService.java service/
cp CiliInviteCodeServiceImpl.java service/impl/
cp CiliInviteCodeController.java controller/

# src/main/resources/mapper/
cp CiliInviteCodeMapper.xml mapper/system/
```

### 2. 复制火山引擎服务代码

```bash
cp VolcanoEngineService.java service/
cp VolcanoEngineServiceImpl.java service/impl/
```

### 3. 配置火山引擎密钥

在 `application.yml` 中添加：

```yaml
volcano:
  access-key: your_access_key
  secret-key: your_secret_key
  endpoint: open.volcengineapi.com
  region: cn-north-1
```

### 4. 添加菜单

在数据库中执行：

```sql
-- 添加CiliAI菜单目录
INSERT INTO sys_menu VALUES(2000, 'CiliAI管理', 1, 10, 'cili', 'cili/index', '', 'cili', 1, 0, 'M', '0', '0', '', 'cilixxx', 'admin', NOW(), '', NULL, 'CiliAI业务模块');

-- 添加邀请码管理菜单
INSERT INTO sys_menu VALUES(2001, '邀请码管理', 2000, 1, 'invitecode', 'system/invitecode/index', '', 'invitecode', 1, 0, 'C', '0', '0', 'cili:invitecode:list', 'user', 'admin', NOW(), '', NULL, '邀请码管理');
```

### 5. 编译并重启

```bash
mvn clean package -DskipTests
java -jar ruoyi-admin.jar
```
