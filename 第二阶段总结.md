## 🎯 立即开始 - 手动启动

由于自动启动遇到沙箱限制，请按以下步骤手动启动：

### 方法一：在 PowerShell 中手动启动（推荐）

```powershell
# 1. 进入项目目录
cd d:\fangtang\RuoYi-Vue-master

# 2. 启动后端（如果遇到日志文件问题，可以跳过日志文件）
java -jar ruoyi-admin/target/ruoyi-admin.jar

# 3. 等待看到 "Started RuoYiApplication" 消息
# 4. 保持终端打开
```

### 方法二：跳过日志文件

如果仍然有日志文件问题，可以：

```powershell
# 创建日志目录
New-Item -ItemType Directory -Force -Path "D:/ruoyi/logs"

# 使用管理员权限运行
Start-Process powershell -Verb RunAs -ArgumentList "-Command cd d:\fangtang\RuoYi-Vue-master; java -jar ruoyi-admin/target/ruoyi-admin.jar"

# 或者直接双击运行 jar 文件
explorer d:\fangtang\RuoYi-Vue-master\ruoyi-admin\target
```

### 方法三：使用 Maven 启动

```powershell
cd d:\fangtang\RuoYi-Vue-master
mvn spring-boot:run -pl ruoyi-admin
```
